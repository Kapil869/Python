# Secure Password Generator
A Python-based utility that generates strong, randomized passwords based on user-defined criteria.

### Key Features:
* **Customizable Complexity:** Allows users to include/exclude uppercase, numbers, and special characters.
* **Variable Length:** Generates passwords of any length to meet specific site requirements.
* **Security Focused:** Uses Python's `random` and `string` modules for reliable character selection.

### Tech Stack:
* **Language:** Python
* **Concepts:** String Manipulation, Logic Loops, Randomization

### How to use:
1. Run the script.
2. Enter the desired password length.
3. The script will output a secure, randomized string.
